# Text Generation with Markov Chains

## Prodigy InfoTech – Generative AI Internship | Task 3

### Task Description
Implement a simple text generation algorithm using Markov chains. This task involves creating a statistical model that predicts the probability of a character or word based on the previous one(s).

### Overview
This project builds a word-level Markov chain from a body of text and uses it to probabilistically generate new, similarly-styled text. No external libraries are required — it's implemented using core Python only.

### Tech Stack
- Python 3.8+ (standard library only)
